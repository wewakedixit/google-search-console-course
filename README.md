# Google Search Console Course for Beginners

A plain-English beginner course on Google Search Console for non-technical users.

The course is based on Google's official Search Console documentation and includes:

- 10 beginner lessons
- Plain-English SEO terms
- Practical examples
- Setup, monthly review, and page improvement checklists
- A quick quiz
- Links to the official Google sources used

## Local Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The production output is generated in `dist/`.
